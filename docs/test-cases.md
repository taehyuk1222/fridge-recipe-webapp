# Fridge Recipe 기능 검증 테스트 케이스

본 문서는 Fridge Recipe 웹앱의 주요 기능이 요구사항에 맞게 정상 작동하는지 확인하기 위한 기능 검증 문서입니다.

## 1. 식재료 관리 기능 테스트

### 식재료 등록 테스트
- [ ] 식재료명, 수량, 유통기한을 입력 후 등록 버튼을 클릭하면 나의 냉장고 목록에 즉시 추가되는가?
- [ ] 필수 입력 항목 누락 시 등록이 제한되거나 안내 메시지가 표시되는가?

### 식재료 수정 테스트
- [ ] 등록된 식재료의 수정 버튼을 클릭하여 수량이나 유통기한을 변경할 수 있는가?
- [ ] 수정 내용이 저장된 후 냉장고 목록과 추천 결과가 즉시 갱신되는가?

### 식재료 삭제 테스트
- [ ] 등록된 식재료의 삭제 버튼을 클릭 시 목록에서 정상적으로 제거되는가?
- [ ] 삭제 후 추천 레시피 결과가 삭제된 재료를 제외하고 즉시 갱신되는가?

## 2. 유통기한 및 상태 표시 테스트

### D-day 계산 테스트
- [ ] 선택한 유통기한과 현재 날짜를 기준으로 D-day가 정확히 계산되어 표시되는가?

### 소비 기한 임박 식재료 표시 테스트
- [ ] 기한 만료, 오늘까지, D-1 ~ D-3에 해당하는 재료가 '소비 기한 임박' 영역에 시각적으로 강조되어 표시되는가?

## 3. 추천 레시피 기능 테스트

### 기한 만료 재료 추천 제외 테스트
- [ ] 기한이 지난 재료가 추천 레시피 계산(가용 재료)에서 확실하게 제외되는가?

### 보유 재료 기반 추천 테스트
- [ ] 보유 중인 재료와 레시피의 요구 재료(ingredients)가 최소 1개 이상 일치하는 레시피가 동적으로 출력되는가?

### 부족 재료 표시 테스트
- [ ] 추천 레시피 카드에 '보유 중인 재료'와 '추가 구매가 필요한 부족 재료'가 명확히 구분되어 표시되는가?

### 부족 재료 3개 이하 조건 테스트
- [ ] 레시피 조리를 위해 필요한 부족 재료가 4개 이상인 레시피는 추천 목록에서 제외되는가?

### 추천 점수 산출 범위 테스트
- [ ] 현재 냉장고 상태 기준 조리 적합도를 나타내는 추천 점수가 50점~100점 범위 내에서 정상 표시되는가?
- [ ] 일치하는 재료가 많고 임박 재료가 포함될수록 점수가 더 높게 산출되는가?

## 4. 부가 기능 및 안정성 테스트

### 레시피 상세보기 모달 테스트
- [ ] 추천 카드의 상세보기 버튼 클릭 시 모달이 열리며 상세 정보(조리 순서 등)가 정상 출력되는가?
- [ ] 닫기 버튼, 배경 클릭, ESC 키 입력을 통해 모달이 정상적으로 닫히는가?

### 데이터 유지(localStorage) 테스트
- [ ] 식재료를 등록/수정/삭제한 후 브라우저를 새로고침해도 데이터와 추천 상태가 그대로 유지되는가?

### 외부 레시피 링크 제거 확인
- [ ] 외부 레시피 사이트로 연결되는 링크가 남아있지 않고 내부 모달로만 처리되는가?
